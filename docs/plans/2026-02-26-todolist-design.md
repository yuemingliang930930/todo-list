# 待办事项应用设计文档

**项目名称**: Todo List App
**版本**: v1.0
**日期**: 2026-02-26
**技术栈**: React + TypeScript + Vite + Tailwind CSS

---

## 1. 项目概述

一个功能完整的待办事项列表应用，支持多列表分组、标签系统、主题切换和数据本地持久化。

---

## 2. 功能需求

### 2.1 核心功能

| 功能 | 描述 |
|------|------|
| 待办事项管理 | 添加、编辑、删除、标记完成 |
| 多列表分组 | 支持创建多个列表（如"工作"、"生活"） |
| 标签系统 | 每个待办可添加多个标签 |
| 主题切换 | 支持明/暗主题切换 |
| 数据持久化 | localStorage本地存储 |

### 2.2 用户交互

- 点击待办可切换完成状态
- 双击待办可编辑内容
- 右键/长按显示操作菜单（删除、编辑、添加标签）
- 拖拽调整顺序（可选）

## 4. 数据模型

### 4.1 待办事项 (TodoItem)

```typescript
interface TodoItem {
  id: string;           // 唯一标识
  content: string;     // 内容
  completed: boolean;  // 是否完成
  listId: string;      // 所属列表ID
  tags: string[];      // 标签数组
  createdAt: number;   // 创建时间
  updatedAt: number;   // 更新时间
}
```

### 4.2 列表 (TodoList)

```typescript
interface TodoList {
  id: string;
  name: string;
  color: string;       // 列表颜色
  createdAt: number;
}
```

### 4.3 标签 (Tag)

```typescript
interface Tag {
  id: string;
  name: string;
  color: string;
}
```

## 5. UI/UX 设计

### 5.1 布局结构

```
┌─────────────────────────────────────────────────────────┐
│  Header (主题切换 + 搜索)                               │
├──────────────┬──────────────────────────────────────────┤
│              │  当前列表标题                            │
│   Sidebar    ├──────────────────────────────────────────┤
│  (列表管理)  │                                          │
│              │  待办事项列表                            │
│  - 列表1    │  ┌────────────────────────────────────┐  │
│  - 列表2    │  │ ☐ 待办事项1 [标签1] [标签2]        │  │
│  - 列表3    │  │ ✓ 待办事项2                         │  │
│              │  │ ☐ 待办事项3                         │  │
│  + 新建列表  │  └────────────────────────────────────┘  │
│              │                                          │
│  标签管理    │  + 添加新待办                            │
└──────────────┴──────────────────────────────────────────┘
```

### 5.2 响应式断点

| 设备 | 宽度 | 布局 |
|------|------|------|
| 移动端 | < 768px | 侧边栏隐藏，汉堡菜单切换 |
| 平板 | 768px - 1024px | 侧边栏可折叠 |
| 桌面 | > 1024px | 侧边栏常显 |

### 5.3 主题设计

**亮色主题**:
- 背景: `#FFFFFF`
- 文字: `#1F2937`
- 主色: `#3B82F6` (蓝色)
- 边框: `#E5E7EB`

**暗色主题**:
- 背景: `#1F2937`
- 文字: `#F9FAFB`
- 主色: `#60A5FA` (浅蓝色)
- 边框: `#374151`
